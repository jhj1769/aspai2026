Summer School Program Details

Day 1 – Thursday, August 27th, 2026

09:00 – 09:30 Opening & Welcome
Introduction to APPI Summer School
09:30 – 11:00 Session 1: Foundations of Process Mining
Introduction to process mining concepts, event logs, and process discovery techniques
11:00 – 11:30 Coffee Break
11:30 – 13:00 Session 2: Process Discovery Algorithms
Alpha Miner, Inductive Miner, and Heuristics Miner
13:00 – 14:00 Lunch Break
14:00 – 15:30 Session 3: Hands-on Workshop (PM4Py)
Practical exercises using PM4Py Python library
15:30 – 16:00 Coffee Break
16:00 – 17:30 Session 4: Conformance Checking
Token-based replay, alignments, and fitness metrics
18:30 – Welcome Dinner

Day 2 – Friday, August 28th, 2026

09:00 – 10:30 Session 5: Performance Analysis
Performance spectrum, bottleneck analysis, and KPIs
10:30 – 11:00 Coffee Break
11:00 – 12:30 Session 6: Hands-on Workshop (Celonis)
Industry-grade process mining with Celonis platform
12:30 – 13:30 Lunch Break
13:30 – 15:00 Session 7: Object-Centric Process Mining (OCPM)
Introduction to OCPM and multi-object event logs
15:00 – 15:30 Coffee Break
15:30 – 17:00 Session 8: Predictive Process Monitoring & AI
Machine learning in process mining, predictive analytics
17:00 – 17:30 Closing Ceremony
Certificate distribution and group photo

Topics Covered

Process Discovery

  • Event log preprocessing
  • Alpha Miner algorithm
  • Inductive Miner
  • Heuristics Miner
  • Directly-Follows Graphs

Conformance & Performance

  • Token-based replay
  • Alignments
  • Fitness & precision metrics
  • Performance analysis
  • Bottleneck detection

Advanced Topics

  • Object-Centric Process Mining
  • Predictive process monitoring
  • Machine learning integration
  • Responsible process mining
  • Industry applications

Tools & Technologies

Participants will gain hands-on experience with:

  • PM4Py: Open-source Python library for process mining
  • Celonis: Industry-leading process mining platform
  • ProM: Extensible framework for process mining research
  • bupaR: R-based process mining framework
